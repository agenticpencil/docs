Perform a comprehensive audit of your website's content to identify high-performing pages, underperforming content, and optimization opportunities.

<RequestExample>

```bash cURL
curl -X POST "https://api.agenticpencil.com/v1/content/audit" \
  -H "Authorization: Bearer ap_your_api_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "domain": "yoursite.com",
    "sitemap_url": "https://yoursite.com/sitemap.xml",
    "max_pages": 100
  }'
```

```python Python
import requests

url = "https://api.agenticpencil.com/v1/content/audit"
headers = {
    "Authorization": "Bearer ap_your_api_key_here",
    "Content-Type": "application/json"
}
data = {
    "domain": "yoursite.com",
    "sitemap_url": "https://yoursite.com/sitemap.xml",
    "max_pages": 100
}

response = requests.post(url, headers=headers, json=data)
result = response.json()
```

```javascript Node.js
const axios = require('axios');

const response = await axios.post(

    }
  }
);
```

## Authentication


## Request Body

<ParamField body="domain" type="string" required>
  Your website domain (e.g., "yoursite.com")
</ParamField>

<ParamField body="sitemap_url" type="string">
  URL to your XML sitemap (if not provided, we'll attempt to find it automatically)
</ParamField>

<ParamField body="max_pages" type="integer">
  Maximum number of pages to audit (1-500, default: 100)
</ParamField>

## Content Score Breakdown

<AccordionGroup>
  <Accordion title="Score Range 80-100: Excellent">
    **High-performing content** that's driving significant organic traffic and ranking well for target keywords. These pages are your content champions and should be used as templates for future content.
  </Accordion>

  <Accordion title="Score Range 60-79: Good">
    **Solid performing content** with room for optimization. Usually requires minor improvements like better internal linking, updated meta tags, or content freshening.
  </Accordion>

  <Accordion title="Score Range 40-59: Needs Improvement">
    **Underperforming content** that requires significant optimization or refreshing. These pages have potential but need strategic improvements to reach their ranking potential.
  </Accordion>

  <Accordion title="Score Range 0-39: Poor">
    **Low-performing content** that may need complete rewriting, consolidation, or removal. These pages often have thin content, poor keyword targeting, or technical issues.
  </Accordion>
</AccordionGroup>

## Best Practices

<Tip>
  **Regular Audits**: Run content audits quarterly to track improvements and identify new optimization opportunities.
</Tip>

<Tip>
  **Focus on High-Impact Pages**: Prioritize optimizing pages with high traffic potential and medium-to-low current performance.
</Tip>

<Tip>
  **Address Cannibalization**: Pages competing for the same keywords can hurt each other's performance. Use the insights to identify and resolve these issues.
</Tip>

<Warning>
  Ensure your sitemap is up-to-date and accessible. Outdated sitemaps will result in incomplete audits.
</Warning>
