Research relevant keywords for your domain and topic, complete with search volume, competition analysis, and opportunity scoring.

<RequestExample>

```bash cURL
curl -X POST "https://api.agenticpencil.com/v1/keywords/research" \
  -H "Authorization: Bearer ap_your_api_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "domain": "example.com",
    "topic": "sustainable fashion",
    "country": "US",
    "limit": 20
  }'
```

```python Python
import requests

url = "https://api.agenticpencil.com/v1/keywords/research"
headers = {
    "Authorization": "Bearer ap_your_api_key_here",
    "Content-Type": "application/json"
}
data = {
    "domain": "example.com",
    "topic": "sustainable fashion",
    "country": "US",
    "limit": 20
}

response = requests.post(url, headers=headers, json=data)
result = response.json()
```

```javascript Node.js
const axios = require('axios');

const response = await axios.post(
  'https://api.agenticpencil.com/v1/keywords/research',

  },

    headers: {
      'Authorization': 'Bearer ap_your_api_key_here',

    }
  }
);
```

</RequestExample>

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token using your AgenticPencil API key
</ParamField>

## Request Body

<ParamField body="domain" type="string" required>
  Your website domain (e.g., "example.com")
</ParamField>

<ParamField body="topic" type="string" required>
  The topic or niche you want to research keywords for
</ParamField>

<ParamField body="country" type="string" required>
  Two-letter country code for geo-targeted results (e.g., "US", "UK", "DE")
</ParamField>

<ParamField body="limit" type="integer">
  Maximum number of keywords to return (1-100, default: 50)
</ParamField>

## Response

<ResponseField name="status" type="string">
  Status of the request ("success" or "error")
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="data object">
    <ResponseField name="keywords" type="array">
      Array of keyword objects with research data
      <Expandable title="keyword object">
        <ResponseField name="keyword" type="string">
          The keyword phrase
        </ResponseField>
        <ResponseField name="search_volume" type="integer">
          Average monthly search volume
        </ResponseField>
        <ResponseField name="competition" type="string">
          Competition level: "low", "medium", or "high"
        </ResponseField>
        <ResponseField name="cpc" type="number">
          Cost per click in USD
        </ResponseField>
        <ResponseField name="trend" type="string">
          Search trend: "increasing", "decreasing", or "stable"
        </ResponseField>
        <ResponseField name="difficulty" type="integer">
          SEO difficulty score (1-100, higher is more difficult)
        </ResponseField>
        <ResponseField name="opportunity_score" type="integer">
          AgenticPencil opportunity score (1-100, higher is better)
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="total_found" type="integer">
      Total number of keywords found
    </ResponseField>
    <ResponseField name="credits_used" type="integer">
      Number of credits consumed by this request
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="credits_remaining" type="integer">
  Your remaining credit balance
</ResponseField>

## Tips for Better Results

<Tip>
  **Be Specific**: More specific topics yield better, more targeted keyword suggestions than broad, generic terms.
</Tip>

<Tip>
  **Use Your Domain**: Including your actual domain helps our algorithm provide more relevant keyword suggestions based on your site's context.
</Tip>

<Tip>
  **Consider Geography**: Different countries have different search patterns. Always specify the country code for your target audience.
</Tip>
